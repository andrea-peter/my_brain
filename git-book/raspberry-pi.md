# Raspberry Pi

This pages refers to the Raspberry Pi 3 model B

## Burn image to SD card

### Prerequisites

```
sudo apt install rpi-imager
```

### Use Raspberry Pi imager

rpi-imager is a GUI program that downloads and burns images on media

```
rpi-imager
```
